## 系统概述

DMS（Document & Device Management System）设备管理系统是一套面向制药/医疗设备行业的综合管理平台，主要功能包括：

- **设备台账管理**：设备基本信息、维护记录、校准记录、维修记录
- **文档全生命周期管理**：文档上传、审批流程、版本控制、生效/归档
- **备件库存管理**：备件入库、消耗追踪、库存预警、成本统计
- **借阅管理**：文档借阅与归还追踪
- **电子签名**：符合 21 CFR Part 11 要求的签名验证与审计
- **审批流程**：多级审批支持
- **提醒中心**：待办事项集中展示
- **用户权限管理**：基于角色的访问控制（RBAC），7 种角色 22 项权限

## 快速开始

### 1. 安装与启动

```bash
# 进入项目目录
cd D:\EquipmentManagement

# 启动服务器（开发模式）
python app.py
```

服务启动后访问：`http://localhost:5000`

### 2. 默认账号

| 用户名 | 密码 | 角色 |
|--------|------|------|
| admin | admin123 | 管理员 |
| user | user123 | 设备工程师 |

> ⚠️ **安全提示**：生产环境请务必修改默认密码！

### 3. 首次登录

1. 访问登录页面
2. 输入用户名/密码
3. 点击登录按钮
4. 登录成功后自动跳转至数据看板

### 设备管理

**菜单位置**：设备管理

#### 1. 设备列表

查看所有设备信息：设备编号、名称、型号、位置、状态。

**状态类型**：
| 状态 | 说明 |
|------|------|
| active | 运行中 |
| maintenance | 维护中 |
| inactive | 停用 |
| retired | 报废 |
| debug | 调试 |
| standby | 待机 |
| repair | 维修 |

#### 2. 新增设备

1. 点击「新增设备」按钮
2. 填写设备信息（编号、名称、型号、位置）
3. 点击保存

> ⚠️ 设备编号不能重复

#### 3. 设备详情

点击设备进入详情页，可查看：基本信息、关联文档、维护记录、校准记录、变更历史。

#### 4. 设备变更

变更到「停用」或「报废」状态需审批：
1. 点击「申请变更」→ 选择目标状态 → 填写原因
2. 提交后等待审批
3. 审批通过后生效

---

### 维护与维修

**菜单位置**：设备管理 → 进入具体设备 → 维护记录

#### 维护计划

可以创建周期性的维护/校准/巡检计划：
- **校准**：确保设备精度合规
- **保养**：预防性维护保养
- **巡检**：定期巡检检查

**固定周期选项**：7天（每周）、30天（每月）、90天（每季度）、180天（每半年）、365天（每年）

#### 维护记录

每次执行维护后，记录：
- 执行日期、执行人
- 维护结果（合格/不合格/待处理）
- 消耗的备件（自动扣减库存）
- 备注说明

#### 维修记录

设备故障时：
1. 进入设备详情 →「维修记录」→「新建维修」
2. 填写故障描述、维修内容
3. 关联消耗的备件
4. 提交（自动触发电子签名确认）

#### 维护历史

可按类型和年份筛选历史维护记录，支持导出 Excel。

### 备件库存管理

**菜单位置**：备件库存管理

#### 1. 备件列表

查看所有备件库存，支持：
- 按名称/编码/型号/品牌搜索
- 按分类筛选
- 按库存状态筛选（正常/库存不足/缺货/超量）

**库存状态说明**：
| 状态 | 说明 |
|------|------|
| 正常 | 库存量在安全范围内 |
| 库存不足 | 低于安全库存下限 |
| 缺货 | 当前库存为 0 |
| 库存超量 | 超过安全库存上限 |

#### 2. 新增备件

1. 点击「新增备件」按钮
2. 填写：名称、分类、规格型号、单位、品牌
3. 设置安全库存上下限
4. 填写供应商信息（可选）
5. 保存

#### 3. 备件入库

1. 点击备件行末的入库按钮（绿色 + 图标）
2. 填写入库数量、采购单价
3. 可选填写批次号和入库日期
4. 确认入库

> 入库后系统自动更新库存量和加权平均价。

#### 4. 备件消耗

1. 点击备件行末的消耗按钮（红色 - 图标）
2. 填写消耗数量
3. 填写消耗原因
4. 确认消耗

> 消耗单价自动取当前加权平均价，消耗后自动扣减库存。

#### 5. 库存预警

系统自动监控库存状态：
- 低库存预警：库存低于安全下限
- 缺货预警：库存为 0
- 超量预警：库存超过安全上限
- 呆滞预警：长期未消耗的备件

预警可在「预警中心」页面集中查看和处理。

#### 6. 成本统计

按时间范围、设备维度查看备件消耗成本分析。

#### 7. 数据导出

备件列表、入库记录、消耗记录均支持 Excel 导出。

### 文档中心

**菜单位置**：文档中心

#### 1. 全部文档

查看所有文档，支持按设备、文档类型、状态、上传人筛选。

**文档类型**：
| 类型 | 说明 |
|------|------|
| equipment_history | 设备履历表 |
| urs | URS |
| dq | DQ报告 |
| iq | IQ报告 |
| oq | OQ报告 |
| pq | PQ报告 |
| calibration | 校准记录 |
| maintenance | 维护记录 |
| deviation | 偏差报告 |
| change | 变更记录 |
| drawing | 图纸 |
| manual | 手册 |

**文档状态**：起草 → 审批中 → 生效 → 变更中 → 归档 / 作废

#### 2. 上传文档

1. 点击「上传文档」→ 选择关联设备
2. 选择文档类型、填写名称和版本
3. 上传文件（支持 PDF、Word、Excel、图片，最大 50MB）
4. 提交后可选择提交审批

#### 3. 文档下载 / 借阅

点击文件名下载；点击「借阅」可追踪借出/归还状态。

#### 4. 文档审批

管理类角色可在顶部看到待审批数量提醒：
1. 查看待审批文档详情
2. 选择「通过」或「驳回」
3. 填写审批意见
4. **电子签名确认**（必须）

### 电子签名

**菜单位置**：电子签名

符合 21 CFR Part 11 要求的电子签名功能。

#### 签名验证

在以下场景触发电子签名：
- 文档审批（审批确认）
- 维护记录提交（执行确认）
- 维修记录提交（执行确认）
- 设备状态变更（变更确认）
- QA 审核（审核确认）

**验证方式**：弹窗中输入当前用户的用户名和密码进行二次验证。

#### 安全机制

- 密码重验证（用户名 + 密码双要素）
- 防代签（验证当前登录用户与签名用户一致）
- 3 次失败后锁定 5 分钟
- 签名记录不可修改/删除（审计要求）
- 服务器时间戳防篡改

#### 签名记录

可查看所有签名历史，按业务类型和签名含义筛选。

## 常见问题

### Q1: 忘记密码怎么办？

**答**：在登录页点击「忘记密码」提交重置请求，联系管理员处理。或直接联系管理员在「用户管理」中重置。

### Q2: 为什么看不到某个菜单？

**答**：当前角色没有该菜单权限，联系管理员开通。

### Q3: 文档上传失败？

**答**：检查：1. 文件是否超过 50MB；2. 格式是否为 pdf/doc/docx/xls/xlsx/jpg/png；3. 网络是否正常。

### Q4: 审批流程需要几步？

**答**：目前为单级审批，由拥有 `document_approval` 权限的角色审批即可。审批时需要电子签名确认。

### Q5: 设备状态变更需要审批吗？

**答**：变更到「停用」或「报废」需要审批，其他状态可直接变更。

### Q6: 电子签名锁定后怎么办？

**答**：3 次密码错误后锁定 5 分钟，5 分钟后自动解锁。或联系管理员通过 API 手动解锁。

### Q7: 备件消耗后库存未减少？

**答**：检查消耗操作是否成功提交。在「消耗记录」页面可查看消耗历史。

### Q8: 如何导出数据？

**答**：设备列表、文档列表、备件列表、入库记录、消耗记录、维护历史等均支持 Excel 导出，在各列表页面点击「导出」按钮即可。
