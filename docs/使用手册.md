## 系统概述

DMS（Document & Device Management System）设备管理系统是一套面向制药/医疗设备行业的综合管理平台，主要功能包括：

- **设备台账管理**：设备基本信息、维护记录、校准记录
- **文档全生命周期管理**：文档上传、审批流程、生效/归档
- **借阅管理**：文档借阅与归还追踪
- **审批流程**：多级审批支持
- **提醒中心**：待办事项集中展示
- **用户权限管理**：基于角色的访问控制（RBAC）

### 技术栈

| 组件 | 技术 |

| 数据库 | SQLite |
| 前端模板 | Jinja2 + Bootstrap 5 |

| 图标库 | Lucide Icons |

### 设备管理

**菜单位置**：设备管理

**子功能**：

#### 1. 设备列表

查看所有设备信息，包括：
- 设备编号
- 设备名称
- 型号
- 位置
- 状态

**状态类型**：
| 状态 | 说明 |
|------|------|
| active | 运行中 |
| maintenance | 维护中 |
| inactive | 停用 |
| retired | 报废 |
| debug | 调试 |
| standby | 待机 |
| repair | 维修 |

#### 2. 新增设备

**操作步骤**：
1. 点击「新增设备」按钮
2. 填写设备信息（编号、名称、型号、位置）
3. 点击保存

> ⚠️ 设备编号不能重复

#### 3. 设备详情

点击设备列表中的任意设备进入详情页，可查看：
- 基本信息
- 关联文档
- 维护记录
- 校准记录
- 变更历史

#### 4. 设备变更

**适用角色**：admin、qa_manager、production_supervisor

当设备状态需要变更为「停用」或「报废」时：
1. 点击设备详情页的「申请变更」按钮
2. 选择变更类型和目标状态
3. 填写变更原因
4. 提交申请
5. 待审批通过后生效

#### 5. 设备校准

**适用角色**：equipment_engineer、metrology_engineer

**操作步骤**：
1. 进入设备详情页
2. 点击「校准记录」标签
3. 点击「新增校准」按钮
4. 填写校准信息（校准日期、校准机构、结果等）
5. 保存记录

#### 6. 设备维护

**操作步骤**：
1. 进入设备详情页
2. 点击「维护记录」标签
3. 点击「新增维护」按钮
4. 填写维护信息（维护类型、内容、结果等）
5. 保存记录

1. 点击「上传文档」按钮
2. 选择关联设备
3. 选择文档类型
4. 填写文档名称和版本
5. 上传文件（支持 PDF、Word、Excel、图片等）
6. 填写备注信息
7. 点击提交

- 点击文件名可预览

- 下载次数会被记录

2. 点击「借阅」按钮

4. 借阅成功后可下载文档

**适用角色**：admin、qa_manager、production_supervisor、validation_engineer

管理类角色可在顶部看到待审批数量提醒，点击进入审批页面：
1. 查看待审批文档列表
2. 点击文档查看详情
3. 选择「通过」或「驳回」
4. 如需可填写审批意见

## 常见问题

### Q1: 忘记密码怎么办？

**答**：请联系系统管理员（admin），由管理员在「用户管理 - 密码重置」中手动重置密码。

### Q2: 为什么看不到某个菜单？

**答**：可能原因：
1. 当前角色没有该菜单的访问权限
2. 该菜单已被管理员禁用
3. 需要联系管理员开通权限

### Q3: 文档上传失败怎么办？

**答**：检查以下内容：
1. 文件大小是否超过 50MB
2. 文件格式是否在允许列表中（pdf, doc, docx, xls, xlsx, jpg, jpeg, png）
3. 网络连接是否正常

### Q4: 审批流程需要几步？

**答**：目前文档审批流程为单级审批，由管理员（admin）审批即可生效。

### Q5: 设备状态变更需要审批吗？

**答**：是的，当设备状态变更为「停用」或「报废」时，需要管理员审批。

### Q6: 如何查看审计日志？

**答**：以 admin 身份登录后，进入「用户管理 - 审计日志」查看。
